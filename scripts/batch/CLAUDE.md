# Shell Batch Scripts

| Script | Purpose |
|--------|---------|
| `run_ood_batch.sh` | Run rule-based SGE on OOD datasets |
| `run_ood_llm_batch.sh` | Run LLM-enhanced SGE on OOD datasets (full set) |
| `run_ood_llm_3datasets.sh` | Run LLM-enhanced SGE on 3 failed OOD datasets |

All scripts `cd` to project root before execution.
